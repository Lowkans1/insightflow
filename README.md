# 📊 InsightFlowKDS (Decision Support System)

InsightFlow is a product-minded, data-driven Decision Support System (KDS) designed to bridge the gap between complex data analytics and strategic business decisions. Built with a modern web ecosystem, it empowers businesses to track, visualize, and analyze critical operational metrics efficiently.

## 🎯 Product & Business Impact (The MIS Perspective)
As a Management Information Systems (MIS) graduate, this project was architected around a clear business problem:
- **The Problem:** Organizations often struggle to centralize scattered data, leading to delayed decision-making and missed strategic insights.
- **The Solution:** InsightFlow centralizes frontend data visualization, transforming raw metrics into real-time, actionable business intelligence (BI) and decision-making support.

## 🧠 AI-Accelerated Engineering Workflow
The entire development lifecycle of InsightFlow heavily embraced modern, AI-native practices to increase development velocity and guarantee reliability:
- **Rapid Prototyping:** Leveraged LLMs to efficiently scaffold UI components and structure core application logic.
- **Verification & Testing:** Used AI tools to automate test case generation, ensuring the decision-making logic is reliable before deployment.
- **Pragmatic Problem Solving:** Solved complex asynchronous data fetching issues by pairing with AI assistants to debug from first principles.

## 🛠️ Tech Stack & Tools
- **Frontend:** React, JavaScript, Tailwind CSS
- **Tools:** Git, GitHub Copilot, ChatGPT (Prompt Engineering)
